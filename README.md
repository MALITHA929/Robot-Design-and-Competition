# Robot-Design-and-Competition
Designing a Robot to Perform Predefined Tasks

🤖 Autonomous Task-Based Robot 🚀

This project was developed for the Robot Design and Competition module in Semester 03 🎓. The objective was to design and build an autonomous robot capable of performing a series of predefined tasks, including:

🎨 Color Detection
➖ Line Following
🛣️ Dashed-Line Following
➡️ Arrow Following
📦 Box Picking
⛰️ Ramp Climbing

The project was completed by a team of four students 👥 between September 2025 and December 2025 📅.

Throughout the development process, our team successfully passed the mid-term evaluation ✅ and delivered a strong performance in the final evaluation 🏆. As a result, all team members achieved an A grade ⭐ for the module.

💡 This repository contains the project documentation, design files, source code, and other resources related to the development of the robot.
